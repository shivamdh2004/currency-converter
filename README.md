# Currency Converter App 💱

A simple and responsive currency converter application built using **React**, **Vite**, and **Tailwind CSS**. This project demonstrates API handling, state management, and modern front-end development practices — ideal for showcasing my ability to work with real-time data in a clean UI.

## 🔍 Features

- 💱 Real-time currency conversion using an external API
- 🌐 Select from multiple global currencies
- ⚡ Instant conversion updates with state management
- 🎨 Styled with Tailwind CSS for a clean and responsive layout
- 🔄 Built with Vite for fast development experience

## 🚀 Technologies Used

- [React](https://reactjs.org/) — for component-based UI
- [Vite](https://vitejs.dev/) — for fast bundling and dev server
- [Tailwind CSS](https://tailwindcss.com/) — for modern utility-first styling
- [Currency Conversion API] — for fetching live exchange rates

## 🛠️ Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/currency-converter-app.git
cd currency-converter-app
npm install
npm run dev
```
